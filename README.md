NexBA: Intelligent Requirement Management & Collaboration System

NexBA is an AI-powered platform designed to fix the communication gap between non-technical clients and software engineering teams. It translates messy client ideas into structured technical tasks and enforces strict quality control before a project is closed.

## Key Features

-  **AI Task Generation:** Uses OpenAI to automatically turn raw client text into clear User Stories and technical tasks.
-  **Role-Based Workspaces:** Dedicated, secure dashboards for **Clients**, **Business Analysts (BAs)**, and **Developers**.
-  **Evidence-Based Verification:** Developers must upload tangible proof (screenshots/links) before a task can be marked as complete.
-  **Real-Time Chat:** Live messaging and instant notifications powered by Firebase.
-  **Human-in-the-Loop:** BAs act as gatekeepers to review and approve all AI-generated tasks before they go to developers.

##  Technology Stack

- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database & Auth:** Google Firebase (Firestore, Authentication, Storage)
- **Intelligence:** OpenAI API
